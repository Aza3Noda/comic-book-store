# Comic Book Catalog App

This project is a web application designed to manage and display a collection of comic books for sale. It allows users to browse listings, view details, and simulate a marketplace experience.

## Purpose

This project was created as a portfolio piece to demonstrate:
- Frontend development (React)
- Backend/API integration
- UI/UX design principles
- Data management and form handling

## Features

- Display comic book listings
- Add new comics via form
- View details (title, author, price, etc.)
- Responsive UI design

## Tech Stack

### Backend
- Java 24 + Spring Boot 4 — REST API and business logic  
- Spring Data JPA / Hibernate — ORM for database access  
- PostgreSQL — relational database  
- Spring Security — RBAC (user/admin roles)  
- Maven — build and dependency management  

### Frontend
- React 19 (Vite) — fast single-page application  
- Vanilla CSS (custom system) — CSS variables, dark mode, glassmorphism, smooth transitions  
- Inter — modern UI typography  
- Dynamic SVG placeholders — generated fallback covers  

### Integration
- REST API (JSON) — frontend ↔ backend communication  
- Telegram deep linking — pre-filled order messages
